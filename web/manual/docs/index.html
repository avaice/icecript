入出力

print(...string)

引数に指定された文字列を出力する。

              
                  print("Hello, World!")

                  // Hello, World!

input(string)

引数に指定された文字列を使ってユーザーに入力を促す。

                
                    print("こんにちは！" +  input("お名前は？") )
                    // （みさえと入力した場合）
                    // こんにちは！みさえ

readFile(string)

(今のところBrowser onlyですが、Nodeでも対応予定)

引数に指定されたファイルを読み込む。

                
                  var text = readText("hello.txt")
                  print(text)

制御フロー

if

条件式に基づいてコードを実行する。

              
                if (x > 5) {
                  print("x is greater than 5")
                }
                elif (x < 0) {
                  print("x is less than 0")
                }
                else {
                  print("x is 0-5 range")
                }

while

条件式が true の間、コードを繰り返し実行する。

              
                while (x > 0) {
                  print(x)
                  x--
                }

return

関数から値を返す。

              
                fn add(a, b) {
                  return a + b
                }

fn

関数を定義する。

              
                fn greet(name) {
                  print("Hello, " + name + "!")
                }

変数・定数・予約語

var

変数を宣言する。

            
              var x = 10;
              var array = [1, 2, 3]

              print(x) // 10
              print(array[1]) // 2

monkey

true:falseのどちらかを参照する。

            
              print(monkey) // true or false

true

肯定の意味合いを持つ定数。

false

否定の意味合いを持つ定数。

void

何も返すものがない関数が返すべき定数。返す値のない関数でearly returnするときは付けてください。

flag (Browser only)

JavaScriptから指定される、スクリプト実行時の引数。

文字列処理

parse(toType, value)

文字列を解析してデータ型に変換する。

            
              var numStr = "100"
              // int, float, string, number
              var num = parse("int", numStr)

split(text, splitter)

文字列を指定の区切り文字で分割し、配列に格納する。

            
              var fruits = split("apple,banana,orange", ",")
              print(fruits[1]) // banana

数学関数

特筆事項：icecriptは計算の演算子に"-"が存在しない。そのため、5-4をしたい場合は5+-4とする必要がある。

random(number)

0からnum - 1までの乱数を生成する。

            
              var randomNumber = random(100)
